***Equals***
O método equals() é usado para comparar dois objetos para verificar se eles 
são "iguais" em termos de conteúdo ou estado. Por padrão, o método equals() na 
classe Object compara referências de objetos, ou seja, verifica se ambos os objetos 
apontam para o mesmo local na memória.

***List sort***
O método sort() da interface List é usado para ordenar os elementos de uma lista
em ordem crescente ou decrescente, dependendo do comparador fornecido. Ele modifica
a lista original.

***List***
A interface List representa uma coleção ordenada de elementos que podem ser
acessados por um índice. Ela permite elementos duplicados e mantém a ordem de
inserção.

***new ArrayList<>();***
Cria uma nova instância de ArrayList, que é uma implementação da interface List.
ArrayList é uma lista dinâmica que pode crescer conforme necessário.

***new LinkedList<>();***
Cria uma nova instância de LinkedList, que é outra implementação da interface List.
LinkedList é uma lista que usa uma estrutura de dados de lista encadeada, permitindo
inserções e remoções eficientes.

***Complexidade big o***

A notação Big O é usada para descrever a complexidade de tempo ou espaço de um
algoritmo em termos do tamanho da entrada. Ela fornece uma maneira de expressar
como o tempo de execução ou o uso de memória cresce à medida que a entrada aumenta.

***Comparater***

A interface Comparator é usada para definir uma ordem personalizada para objetos. Ela
possui um método compare() que compara dois objetos e retorna um valor inteiro
indicando sua ordem relativa.

***Comparable***

A interface Comparable é usada para definir a ordem natural dos objetos de uma classe.
Ela possui um método compareTo() que compara o objeto atual com outro objeto do
mesmo tipo.
