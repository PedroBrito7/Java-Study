HashSet 
hash é uma implementação da interface Set que utiliza uma tabela hash para armazenar seus elementos. 
Ele não permite elementos duplicados e não mantém a ordem de inserção dos elementos. 
A principal vantagem do HashSet é a sua eficiência na busca, inserção e remoção de elementos,
que geralmente são operações de tempo constante O(1).
