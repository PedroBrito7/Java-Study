##  O que é Pilha (Stack)?

Pilha é uma estrutura de dados que segue a regra:

##  LIFO — Last In, First Out

O último elemento que entra é o primeiro que sai

Exemplo do mundo real:

Pilha de pratos 

Você só tira o prato de cima

Para pegar o de baixo, precisa tirar os de cima antes

##  Operações principais da Pilha

Operação	O que faz

push	Adiciona um elemento no topo

pop	Remove o elemento do topo

peek	Mostra o topo sem remover

isEmpty	Verifica se está vazia

size	Quantidade de elementos
